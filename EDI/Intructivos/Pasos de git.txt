Como subimos cambios a git?
1. Hacer los cambios que necesitamos
En la consola o terminal
1. a- Si hago git status los cambios se ven en ROJO
2. git add . o git add "nombre de archivo"
2. a- Si hago git status los cambios se ven en VERDE
3. git commit -m"Agregando los pasos de git"
4. git push origin main
